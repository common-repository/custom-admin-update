Add Custom Update for your Admin Panel.

== Description ==
Allows administrators to display custom Update in the WordPress admin area. 
This can be helpful for site-wide announcements or important information, 
Using this plugin admin can display Update which can be viewed by all user roles.

This Plugin allows access to edit custom Update by Admin only.
Rest of user roles cannot edit that custom Update but they can view that custom Update.
Using this plugin all users can updated with site latest Update update.

Key Features:

1. Display Custom Update on Admin Area for user roles.

2. With Custom Update Admin can display latest updates regarding Site.

3. All User Roles can View this Custom Update on their Admin Panel.

4. User with Admin Role can only be able to edit and remove Update.

5. All Users of site will be updated information of site with Update.

User Guides:

1. Activate plugin.

2. Admin menu Diplay Custom Updates Option Setting.

3. Admin can add their Update inside text area and save changes.

4. Update will display overall admin panel for all users.

5. Admin can also remove Update using remove button and save changes.

== Installation ==
Follow the below instructions to install this plugin to your WordPress website.

Ex.

1. Upload to the `wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Go to the setting section and open  Custom Admin Update.
4. Add your Css Code and Save Changes.
5. Enjoy!!..
